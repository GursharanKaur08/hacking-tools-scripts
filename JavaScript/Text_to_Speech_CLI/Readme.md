# Text to Speech CLI

A simple script to convert text to speech.

## How to use the script ?

Installations:</br>
- readline-sync</br>
npm install readline-sync
- gtts</br>
npm install gtts</br>

Now when you run the script you will get a voice.mp3 file in your folder.
